Automatically add/update Image attributes(Image Title, Alt Text, Image Caption, Description) from Image Filename.

== Description ==

The plugin provides a funcationality to automatically update Image attributes(Image Title, Alt Text, Image Caption, Description) from Image Filename when upload.

### Features And Options:
* Update Image attributes when image upload
* Options for remove characters, numbers and exta spaces

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "Auto Update Image Attributes From Filename" plugin through the 'Plugins' screen in WordPress
3. Click on Auto Image Attributes on the dashboard.
